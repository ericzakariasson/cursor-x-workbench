# Pinball Game

A lightweight browser pinball game built with plain HTML, CSS, and JavaScript.

## Run locally

1. From this folder, start a static server:
   python3 -m http.server 8000
2. Open `http://localhost:8000` in your browser.

## Controls

- Left flipper: Left Arrow or A
- Right flipper: Right Arrow or D
- Launch: Hold Space to charge, release to launch
- Restart: R
