# Type Anatomy Explorer

An interactive typography playground for design students, built with Next.js.

## Features

- Hover any glyph to inspect animated anatomy guides:
  - x-height
  - ascender line
  - counter markers
  - serif markers
- Compare two Google Fonts side by side
- Edit the sample text live
- Quiz mode with rotating anatomy prompts
- Dark, minimal interface

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```
