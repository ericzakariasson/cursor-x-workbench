# Breakout / Arkanoid

Small no-build browser game built with plain HTML, CSS, and JavaScript.

## Run locally

From the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Controls

- Move paddle: Left/Right Arrow or A/D
- Optional mouse control: move pointer over the game board
- Launch ball: Start button, click canvas, Space, or Enter
- Restart game: Restart button
