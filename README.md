# Retro Minesweeper

Simple retro-style Minesweeper built with plain HTML, CSS, and JavaScript.

Run locally:

1. From this folder, start a static server:
   python3 -m http.server 8000
2. Open http://localhost:8000 in a browser.
3. Left click to reveal cells, right click to place/remove flags.

The game ends when you clear all safe cells or reveal a mine.
