# MS Paint Electron Clone

A lightweight Electron desktop app that recreates core MS Paint style drawing features.

Included tools:
- Pencil, brush, eraser
- Straight line, rectangle, ellipse
- Fill bucket
- Stroke/fill color selection and brush size
- Undo/redo history
- Open image, new canvas, and save PNG export

## Run locally

1. Install dependencies:
   npm install
2. Start the app:
   npm start

## Shortcuts

- Ctrl+Z: Undo
- Ctrl+Shift+Z / Ctrl+Y: Redo
- Ctrl+S: Save PNG
- Ctrl+N: New canvas
