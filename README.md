# Tetris (vanilla HTML/CSS/JS)

Simple self-contained Tetris game with no build step.

How to run:

1. From this repo, start a static file server:
   `python3 -m http.server 8000`
2. Open `http://localhost:8000` in your browser.
3. Click "New Game" and play.

Controls:

- Left/Right arrows: move piece
- Up arrow: rotate
- Down arrow: soft drop
- Space: hard drop
- P: pause/resume
